# crossfit-app

Monorepo para la aplicación Crossfit: API (Express) y Frontend (Vue).

## Estructura

- `api/`: Backend Express
- `frontend/`: App Vue

## Scripts globales

- `npm run dev`: Levanta ambos servicios en desarrollo
- `npm run start:api`: Solo API
- `npm run start:frontend`: Solo frontend

## Desarrollo

1. Clona el repositorio y entra a la raíz.
2. Instala dependencias en la raíz, api y frontend.
